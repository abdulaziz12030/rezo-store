# Rezo Store Starter

هذا مشروع ابتدائي لمتجر **REZO STYLE** مبني باستخدام **Next.js + Tailwind CSS** ومجهز للنشر على **Vercel**.

## ماذا يحتوي؟
- واجهة رئيسية مستوحاة من بنية متجرك الحالي على سلة
- صفحة منتجات
- صفحة منتج فردية
- صفحة سلة مبدئية
- صفحة Checkout مبدئية
- لوحة تحكم أولية
- هيكل قابل للتطوير لاحقًا لربطه مع:
  - Supabase / PostgreSQL
  - بوابة دفع (Moyasar / PayTabs / HyperPay)
  - شركة شحن (Aramex / SPL)
  - تسجيل دخول للعملاء والإدارة

## تشغيل المشروع محليًا
```bash
npm install
npm run dev
```

## النشر على Vercel
1. ارفع الملفات إلى GitHub.
2. اربط المستودع بـ Vercel.
3. نفذ Deploy.

## ملاحظات مهمة
- الصور الحالية **تجريبية** وليست صور منتجاتك الأصلية.
- البيانات الحالية **Mock Data** داخل الملف:
  - `src/lib/data.ts`
- الكود الحالي **ليس متجرًا ديناميكيًا كاملاً بعد**، لكنه قاعدة جاهزة قوية لتطوير المرحلة التالية.

## الملفات المهمة
- `src/app/(store)/page.tsx` الصفحة الرئيسية
- `src/app/(store)/products/page.tsx` صفحة المنتجات
- `src/app/(store)/products/[slug]/page.tsx` صفحة المنتج
- `src/app/(store)/cart/page.tsx` صفحة السلة
- `src/app/(store)/checkout/page.tsx` صفحة إتمام الطلب
- `src/app/admin/page.tsx` لوحة التحكم الأولية
- `src/lib/data.ts` البيانات التجريبية

## المرحلة التالية المقترحة
- ربط Supabase
- إنشاء قاعدة بيانات المنتجات والعملاء والطلبات
- تفعيل سلة حقيقية
- ربط الدفع والشحن
- إضافة تسجيل الدخول
- بناء لوحة تحكم ديناميكية
